# Introduction

Inventory Apoteker System is a web application designed to manage pharmaceutical inventory, track stock levels, and handle sales transactions efficiently. The system helps streamline pharmacy operations and reduce manual record-keeping.

## Key Features

- Medicine inventory management
- Stock in and stock out tracking
- Transaction recording system
- Reporting dashboard for inventory monitoring
- Role-based access control for admin and staff

## Technical Insights

- **Backend**: Developed using Laravel with structured MVC architecture.
- **Frontend**: Built with Bootstrap to ensure responsive and clean UI.
- **Database**: MySQL used for relational data handling and transaction tracking.
- **Validation**: Implemented server-side validation for accurate stock management.

## Challenges and Solutions

- **Stock Accuracy**: Designed automatic stock deduction logic during transactions.
- **Data Consistency**: Structured database relationships to avoid duplication and inconsistencies.

## Outcome

The system successfully digitized pharmacy inventory management, improving efficiency, accuracy, and reporting capabilities.
