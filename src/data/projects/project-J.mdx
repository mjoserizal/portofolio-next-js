# Introduction

PTSP UIN SAIZU adalah sistem layanan terpadu berbasis web yang dikembangkan untuk mendukung proses pelayanan administrasi universitas secara digital, transparan, dan terstruktur.

## Key Features

- Pengajuan layanan administrasi online
- Monitoring status permohonan
- Dashboard rekap data layanan
- Role-based interface untuk admin dan user
- Integrasi API dengan backend system

## Technical Implementation

- **Frontend Development**: Vue.js dengan struktur modular dan scalable.
- **API Communication**: Axios untuk konsumsi REST API.
- **Dynamic Rendering**: Implementasi conditional rendering berdasarkan role pengguna.
- **Responsive UI**: Tailwind CSS untuk memastikan kompatibilitas multi-device.

## Challenges & Solutions

- **Role-Based Access Complexity**
  - Mengatur tampilan dan fitur berdasarkan level akses pengguna.

- **Real-Time Status Updates**
  - Menyesuaikan state dan refresh data agar status layanan selalu terupdate.

## Results & Impact

- Meningkatkan efisiensi pelayanan administrasi universitas.
- Mengurangi antrean layanan manual.
- Meningkatkan transparansi dan monitoring proses layanan.
